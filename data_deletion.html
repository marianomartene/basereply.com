Facebook User Data Deletion

Last updated: October 2025

In accordance with Meta’s platform policies, this page describes how Facebook users can request deletion of data shared with BaseReply via Facebook Login or channel integrations. This page serves as our Data Deletion Instructions URL and can be provided in the Facebook App Dashboard.

1) Remove BaseReply from your Facebook account

  1. Open Facebook and go to Settings & Privacy → Settings.
  2. Select Apps and Websites.
  3. Find BaseReply in the list and click Remove.
  4. Confirm removal. This revokes our access to your Facebook data.

2) Request deletion of data stored by BaseReply

After revoking access, you can ask BaseReply to delete any remaining data that our systems may store about your account. Send an email to privacy@basereply.com with the subject Facebook Data Deletion and include:

  • Your name and contact email.
  • Your Facebook User ID (app-scoped ID is preferred if available).
  • Any relevant details (e.g., the Facebook Page or WhatsApp Business asset connected).

What happens next? We will locate and permanently delete your Facebook-derived data from our systems (including logs where feasible) and confirm by email. If we must retain some data to comply with legal, fraud-prevention, or accounting obligations, we will minimize and isolate it.

3) Automated deletion via callback (for Meta)

BaseReply also supports an endpoint intended for Meta’s automated Data Deletion Request Callback:

https://basereply.com/facebook/data-deletion POST, HTTPS

This endpoint receives a signed request from Meta and initiates deletion associated with the app-scoped user ID. Our response provides a confirmation code and a status URL, for example:

{ "url": "https://basereply.com/deletion-status?code=ABC123", "confirmation_code": "ABC123" }

If you have reached this page directly and are a user (not Meta), please use the email process in section 2.

Scope

This process applies to Facebook data shared with BaseReply via Login or integrations (e.g., Page Inbox, Instagram, WhatsApp Business connections). It does not affect data held by Facebook or other third parties. To remove data held by Facebook, use Facebook’s settings described in section 1.

Contact

For questions about data deletion, contact: privacy@basereply.com
